# Marketing

This directory contains campaign copy and image-generation prompts for promoting `kie-api-python`.

- `linkedin-campaign.md`: four-week, 15-post LinkedIn campaign focused on the CLI + MCP launch.
- `deck.pdf`: rendered campaign deck.
- `deck-viewer.html`: lightweight browser viewer for the rendered deck. Open it locally, or serve `docs/` with GitHub Pages and visit `/marketing/deck-viewer.html`.

The generated source slide PNGs are intentionally ignored under `docs/marketing/slides/` because the PDF is the canonical shareable artifact.
